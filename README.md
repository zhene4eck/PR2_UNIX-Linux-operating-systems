### Инструменты: Linux Ubuntu без графического интерфейса, VirtualBox

**Решенные задачи:**
1. Описан отчет о развертывании Linux OS на VirtualBox и применении базовых функций.
2. Рассмотренные команды, функции и утилиты: 
- создание пользователя и добавление в группу;
- изменение имени виртуальной машины;
- изменение временной зоны;
- вывод названия сетевых интерфейсов;
- получение ip адреса устройства от DHCP-сервера;
- вывод внешнего ip-адрес шлюза (ip) и внутреннего IP-адрес шлюза;
- установка статичных настроек ip, gw, dns;
- обновление системных пакетов;
- использование sudo;
- настройка службы автоматической синхронизации времени;
- установка и использование текстовых редакторов vim, nano, mcedit;
- базовая настройка сервиса SSHD;
- использование утилит top, htop, fdisk, df, du, ncdu;
- работа с системными журналами;
- использование CRON.
